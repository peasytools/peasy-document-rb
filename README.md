# peasy-document

[![Gem Version](https://badge.fury.io/rb/peasy-document.svg)](https://rubygems.org/gems/peasy-document)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

Document conversion library for Ruby — convert between Markdown, HTML, CSV, JSON, and YAML formats. Uses Ruby stdlib (csv, json, yaml), zero external dependencies.

Built from [Peasytools](https://peasytools.com), a free online toolkit.

## Install

```bash
gem install peasy-document
```

## Quick Start

```ruby
require "peasy_document"

# Check version
puts PeasyDocument::VERSION  # => "0.1.1"
```
